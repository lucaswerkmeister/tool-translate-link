Translate Link

This tool generates a link to translate a single message into a certain language on TranslateWiki.net.

On Wikimedia sites and other MediaWiki wikis, you can discover many message keys by adding ?uselang=qqx or &uselang=qqx to the URL, e.g. https://en.wikipedia.org/wiki/Main_Page?uselang=qqx or https://en.wikipedia.org/w/index.php?title=Main_Page&action=history&uselang=qqx. On English Wikipedia, a gadget to reload the current page with the system message names exposed is available (source code).
